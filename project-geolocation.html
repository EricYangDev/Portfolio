Design Notes

What this demonstrates

  • Serious games design: makes a health-related behavior more engaging without relying only on points.
  • Ethical interaction: avoids requiring personal location data for the website demo.
  • Rapid prototyping: models game mechanics in plain JavaScript before investing in a larger mobile or Unity build.
  • Evaluation pathway: future tests could compare motivation, completion rate, and reflection quality across quest types.
